# Decision Support System for Predictive Maintenance

## Project Description

This project is a SQL-based Decision Support System (DSS) developed to monitor patient data and support decision-making in healthcare systems. The system collects patient information, analyzes vital signs, and generates alerts for critical conditions.

## Objectives 

- Monitor patient vital data in real-time  
- Detect abnormal conditions  
- Support decision-making process  
- Store and manage healthcare data  


---

## Technologies Used

- PostgreSQL (Database)
- Python (Backend Logic)
- psycopg2 (Database Connection)
- Tkinter (User Interface)
---

##  Project Structure 

- `hospital_dss_hasan.sql` → Database structure and tables  
- `hospital_monitor.py` → Main monitoring system  
- `test_connection.py` → Database connection test  

---

## How to Run 

1. Install PostgreSQL and create a database  
2. Run the `.sql` file to create tables  
3. Update database connection settings in Python files  
4. Run `hospital_monitor.py`
